## About

**Asignatura:** Estructura y Organizacion de Computadores · UAH GII · Curso 2024-25

Computer organization from transistors to microprocessors: instruction set architecture, CPU pipeline, memory hierarchy, cache design and I/O subsystems.

---

## Topics covered

| Topic | Content |
|-------|---------|
| Number systems | Binary, hex, two's complement, IEEE 754 float |
| Boolean algebra | Logic gates, simplification, Karnaugh maps |
| CPU architecture | ALU, registers, control unit, datapath |
| Instruction sets | RISC vs CISC, addressing modes, ISA design |
| Pipelining | Stages, hazards (data, control, structural), forwarding |
| Memory hierarchy | Cache (direct, associative, set-associative), virtual memory, TLB |
| I/O systems | Interrupts, DMA, bus architecture |
| Assembly | Basic MIPS/x86 assembly, calling conventions |
